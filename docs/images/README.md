# Screenshots

This directory should contain screenshots for the README.

To capture:
1. Run `open /Applications/AIUsageBar.app`
2. Open System Settings → Privacy & Security → Screen Recording → Add Terminal.app
3. Run `screencapture -T 3 docs/images/dashboard.png` (click popover within 3s)

Required files:
- `menubar.png` — menu bar with AIUsageBar icon (800×40)
- `dashboard.png` — Dashboard tab (340×520)
- `budget.png` — Budget tab (340×520)
- `provider.png` — Provider tab (340×520)
